# BMI Calculator

## Objective
This project is a Body Mass Index (BMI) Calculator developed using Python and Tkinter GUI.

## Tools Used
Python
Tkinter

## Steps Performed
1. Created a graphical user interface using Tkinter.
2. Took user input for height and weight.
3. Calculated BMI using the formula:
   BMI = weight / (height^2)
4. Displayed BMI result and health category.

## Outcome
The application allows users to quickly calculate their BMI and understand their health status.
